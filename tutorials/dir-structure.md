# directory structure
### node_modules中react文件夹跟@tarojs/react文件夹是什么关系
React是常说的React库，@tarojs/react是taro框架对React的封装和适配。
### src目录结构
├── app.config.ts   
├── app.less   
├── app.tsx   
├── assets   
├── components      搭建框架的积木         
├── hooks   
├── http   
├── index.html   
├── pages           框架   
├── store   
└── utils   
pages和components是主要结构，其余的都是为了辅助这两个而存在的。
